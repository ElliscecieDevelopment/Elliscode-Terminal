# Elliscode Terminal

## Created by Elliscecie

A fully open-sourced terminal designed and frequently used in Eskrimador Games and LS Gameforge projects.

### Usage Conditions

Modifying Elliscode Terminal is highly encouraged, and even requested. However, modifying or unmodifying this language and referring the source as your own is not permitted. Elliscecie must be credited for the base on every modification.

Examples of this include:
* Claiming you created Elliscode itself
* Leaving a modification uncredited by Elliscecie

Elliscode Terminal is open to all modifications, **if the modifier can handle the complexity of the code**.

Everyone and anyone is always permitted and encouraged to submit their ideas, issues, or custom commands in the *Issues* page or *Pull Requests* page. Please keep issues in the *Issues* only, all else to be placed in the *Pull Requests* page.

Advise that Elliscecie Delteams has every right to suspend or postpone access to this language.
